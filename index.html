Selectores básicos

Los selectores básicos son los más utilizados en CSS. Son los que nos permiten seleccionar elementos HTML por su nombre, id o clase.

Seleccionar por nombre

Para seleccionar un elemento HTML por su nombre, simplemente escribimos el nombre del elemento seguido de las llaves de apertura y cierre.

Por ejemplo, para seleccionar todos los elementos <h1> de la página, escribimos:

p { color: red; }

Seleccionar por id

Dar estilos por id se considera una mala practica o Antipatrón

Para seleccionar un elemento HTML por su id, escribimos el signo de numeral (#) seguido del id del elemento.

Por ejemplo, para seleccionar el elemento con id temario-css, escribimos:

#identidad { color: red; }

☝️

Seleccionar por clase

Para seleccionar un elemento HTML por su clase, escribimos un punto (.) seguido del nombre de la clase.

Por ejemplo, para seleccionar todos los elementos con clase temario, escribimos:

.temario { color: red; }

☝️

Seleccionar por atributo

Para seleccionar un elemento HTML por un atributo, escribimos el nombre del atributo seguido de un signo igual (=) y el valor del atributo entre comillas.

Por ejemplo, para seleccionar todos los elementos <img> que tengan el atributo src con valor logo.png, escribimos:

img[src="assets/dibu.jpg"] { width: 200px; }

☝️
